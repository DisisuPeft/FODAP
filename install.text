Para instalar este proyecto se deben correr los siguientes comandos

1 composer update
2 composer install
3 cp .env.example .env
4 php artisan key:generate
5 se debe configurar el .env y colocar el nombre de la database y password si es que requiere
6 php artisan migrate --seed
7 npm install
8 npm run dev

Correr el sistema e ingresar con el correo
